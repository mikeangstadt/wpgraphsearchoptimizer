== Description ==

This plugin automatically configures your install to use all applicable Facebook OpenGraph tags and exposes the ability to create FBXML elements using shortcodes.  You must have a valid Facebook App ID configured to be associated with the domain of the applicable website for the plug-in to work correctly.  This focuses on the major OpenGraph meta tags around articles (because this is afterall a blogging platform).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `graph-search-optimizer.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
